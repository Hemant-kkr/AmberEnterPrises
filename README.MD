## рҹҡҖ Project Overview

**Amber Safety Enterprises** is a comprehensive e-commerce platform specializing in industrial safety equipment. The application follows a **domain-driven, MVC + feature-based architecture** ensuring clean separation of concerns, scalability, and maintainability.

### Key Highlights

- рҹ”җ **Secure JWT Authentication** with Gmail-based email verification links
- рҹ‘Ҙ **Role-Based Access Control** вҖ” User, Seller, Admin
- рҹҸ—пёҸ **Domain-Driven Architecture** вҖ” Modular, scalable, and maintainable
- рҹӣ’ **Full E-Commerce Flow** вҖ” Browse вҶ’ Cart вҶ’ Checkout
- рҹ“Ұ **Product Management** вҖ” CRUD with image uploads
- рҹ“§ **Transactional Emails** вҖ” Verification & notifications via Nodemailer

```bash

```

### 3пёҸвғЈ Setup Frontend


## рҹ”‘ Environment Variables

### Backend (`server/.env`)

```env
# Server
PORT=5000
NODE_ENV=development

# Database
MONGODB_URI=mongodb+srv://<username>:<password>@cluster.mongodb.net/amber-safety

# JWT
JWT_SECRET=your_super_secret_jwt_key_here
JWT_EXPIRE=7d
JWT_VERIFY_EXPIRE=24h

# Email (Gmail SMTP)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-gmail-app-password

# Client URL (for verification links)
CLIENT_URL=http://localhost:5173
```

### Frontend (`client/.env`)

```env
VITE_API_URL=http://localhost:5000/api
```
